# Brady Inn

A boutique cabin inn website featuring log cabin luxury with refined comfort.



## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Tech Stack

- Next.js 16 (Turbopack)
- TypeScript
- Tailwind CSS 4
- Framer Motion, GSAP, Lenis
- Lucide React
